## 🗂 문서 개요

이 저장소는 Python 학습을 위한 개인 스터디 공간입니다. 각 날짜별로 학습한 내용을 Jupyter Notebook 형식으로 정리두었으며, 연습하고, 아이디어 구상도 일부 포함되어 있습니다.

### ✅ 주요 학습 주제 예시

- 파이썬 기본 문법 (변수, 자료형, 연산자)
- 제어문 (조건문, 반복문)
- 함수 정의 및 활용
- 자료구조 (리스트, 튜플, 딕셔너리, 집합)
- 파일 입출력
- 클래스 정의 및 활용
